var, let, const的区别

var

var声明的变量没有块级作用域而let声明的变量拥有块级作用域，

const

const声明的是常量，声明之后无法对其进行修改， const保证的实际上并不是变量的值不得改变，而是变量指向的那个内存地址不得改动，对于复合类型的数据如对象，数组，变量指向的内存地址保存的只是一个指向实际数据的指针，const只能保证这个指针是固定的

let

在代码块内，使用let命令声明变量之前，该变量都是不可用的，在语法上称为暂时性死区
