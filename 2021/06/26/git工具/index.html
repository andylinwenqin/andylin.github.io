git工具

git工具
git add . 添加到暂存区
git branch 查看分支状态
git checkout master 切换到master分支
git commit -m “” 将暂存区内容添加到本地仓库并加备注
git push 上传远程仓库并合并
git merge login 合并分支
git status 查看仓库状态显示变更文件
git log 查看历史提交记录
git reset 回退版本
git rm 删除工作区文件
git mv 重命名或者移动工作区文件

git branch [name] 本地创建分支git工具
git push origin [name] 本地上传新分支
git branch -d [name] 删除本地分支
git push origin :[name] 删除远程分支
git clone -b dev [] 拉取指定分支代码

github 拉去所有分支

git clone xxx
git branch -r | grep -v ‘->’ | while read remote; do git branch –track “${remote#origin/}” “$remote”; done
git fetch –all
git pull –all
